Install NPM (https://phoenixnap.com/kb/install-node-js-npm-on-windows)
"npm install -g windows-build-tools" (might need to manually do the visual studio launcher, add the v140 tools in the individual components view)
add python to path (if needed) https://datatofish.com/add-python-to-windows-path/ and https://stackoverflow.com/questions/15126050/running-python-on-windows-for-node-js-dependencies
clone repository (might need to install GIT and add to PATH if you want to use command line)
"npm install" (in repository directory)
customize .env
"node index.js" (in repository directory)
